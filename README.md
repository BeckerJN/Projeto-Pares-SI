# Projeto-Pares-SI
Identificação de pares "infectante-infectado" e estimativa de Intervalo Serial (Serial Interval - IS)

DESCRIÇÃO

Código “Pares SI”

O código “Pares SI” destina-se a identificação de pares de infectante-infectado em registros provenientes de sistemas de informação em saúde, e a estimativa de Intervalo Serial (Serial Interval - SI).

Utilize a planilha “Banco_Covid_Modelo” para compreender a estrutura e o desenvolvimento do código “Pares SI”, e como modelo de formatação para outras planilhas a serem analisadas pelo código.

O “Banco_Covid_Modelo” apresenta o registro fictício de 10 casos positivos de COVID_19.

Ao analisar os registros da planilha “Banco_Covid_Modelo” através do código “Pares SI” será identificado dois pares de infectante-infectado e os seguintes resultados serão apresentados:

SI Médio = 12,33

Desvio Padrão = 9,01
